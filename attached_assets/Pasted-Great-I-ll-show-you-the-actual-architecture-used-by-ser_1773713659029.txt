2. Fast Trade Ingestion (100x Faster)

Most people do:

for wallet in wallets:
    GET /trades?user=wallet

This is slow and incomplete.

Instead ingest global trades.

Endpoint:

https://data-api.polymarket.com/trades

Paginate by timestamp.

Example ingestion script:

import requests
import pandas as pd

BASE = "https://data-api.polymarket.com/trades"

def fetch_all_trades():
    offset = 0
    limit = 1000
    trades = []

    while True:
        r = requests.get(BASE, params={
            "limit": limit,
            "offset": offset
        })

        data = r.json()

        if not data:
            break

        trades.extend(data)
        offset += limit

        print("fetched", offset)

    return pd.DataFrame(trades)

Now you store ALL trades once, then query by wallet.

Much faster.

3. Correct PnL Calculation

Polymarket PnL is tricky because selling does not equal profit.

Correct formula:

cost basis
cost = sum(BUY price * size)
realized
sell_pnl = sum(SELL price * size)
final payout
redeem_pnl = winning_shares * $1
total
PnL = sell_pnl + redeem_pnl - cost

Example code:

def calculate_pnl(df):

    buys = df[df.side=="BUY"]
    sells = df[df.side=="SELL"]

    cost = (buys.price * buys.size).sum()
    revenue = (sells.price * sells.size).sum()

    return revenue - cost

Then add redemptions.
