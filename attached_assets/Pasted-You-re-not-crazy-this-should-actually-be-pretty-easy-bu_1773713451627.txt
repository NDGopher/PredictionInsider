4. Working Python Code (Full Trade Pull)

This will pull every trade for a wallet.

import requests
import pandas as pd

ADDRESS = "0x6e82b93eb57b01a63027bd0c6d2f3f04934a752c"

def fetch_trades(address):
    all_trades = []
    offset = 0
    limit = 500

    while True:
        url = f"https://data-api.polymarket.com/trades"
        params = {
            "user": address,
            "limit": limit,
            "offset": offset
        }

        r = requests.get(url, params=params)
        data = r.json()

        if not data:
            break

        all_trades.extend(data)
        offset += limit

    return pd.DataFrame(all_trades)


df = fetch_trades(ADDRESS)

print("Trades:", len(df))
print(df.head())
5. Calculate Position PnL

Each trade has:

price
size
side (BUY/SELL)
market
outcome

PnL formula:

BUY cost = price * shares
SELL revenue = price * shares

PnL = total sells - total buys

Example:

df["notional"] = df["price"] * df["size"]

buys = df[df["side"] == "BUY"]["notional"].sum()
sells = df[df["side"] == "SELL"]["notional"].sum()

pnl = sells - buys

print("Total PnL:", pnl)
6. Breakdown by Market / Category

Merge with markets API:

https://gamma-api.polymarket.com/markets

Example:

markets = requests.get(
    "https://gamma-api.polymarket.com/markets"
).json()

markets_df = pd.DataFrame(markets)

merged = df.merge(
    markets_df,
    left_on="market",
    right_on="id",
    how="left"
)

pnl_by_category = merged.groupby("category")["notional"].sum()
